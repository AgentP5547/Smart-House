# Smart-House
A Smart House project with Arduino.

# Circuit Connections 

### HC-05 Bluetooth Module
* VCC to Arduino 5v
* GND to Arduino GND
* TXD to Arduino RXD
* RXD to Arduino TXD

### Servo 

* Positive (Red) to Arduino 5v
* Negative (Brown) to Arduino GND
* PWM (Orange) to Arduino ~9

### LED or other lights
* Positive to Arduino 7
* Negative to Arduino GND
